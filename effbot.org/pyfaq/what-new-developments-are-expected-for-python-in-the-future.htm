What new developments are expected for Python in the future?

See http://www.python.org/doc/peps/ (dead link) for the Python Enhancement Proposals (PEPs). PEPs are design documents describing a suggested new feature for Python, providing a concise technical specification and a rationale. PEP 1 explains the PEP process and PEP format; read it first if you want to submit a PEP.

New developments are discussed on the python-dev mailing list.
