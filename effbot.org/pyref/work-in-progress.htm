(An Unofficial) Python Reference Wiki (WORK IN PROGRESS)

Work In Progress

This is an experimental PYREF site, intended to replace the infogami site with the same name.

The material on this site consists of a copy of the standard Python Language Reference, broken up in individual pieces, with a distinct URL for each concept, and with extensive semantic hyperlinking between concepts.

The original material has undergone some heavy refactoring, and needs a lot more copy-editing work. Have patience.
