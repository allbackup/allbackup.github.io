sys.settscdump

settscdump( on_flag)

Activate dumping of VM measurements using the Pentium timestamp counter, if onflag is true. Deactivate these dumps if onflag is off. The function is available only if Python was compiled with —with-tsc. To understand the output of this dump, read Python/ceval.c in the Python sources. New in version 2.4.
