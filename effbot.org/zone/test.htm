About the Technology #



The zone.effbot.org subsite is powered by a couple of very
simple Python scripts.  The main script allows me to create and edit
plain text files in a standard browser text area.  Every time I save
the text file, a simple renderer merges the text file with a predefined
template, pushes the result through HTML Tidy
ElementSoup,
and writes the result to disk as a static HTML file.



Other scripts are used to look for broken links, download new and
updated pages and store them in a Perforce Subversion repository, combine them into
a blog page, etc.



I don't plan to release the scripts at this time.  If you need something
similar, I suggest using a real Wiki, such as MoinMoin.
