About the Technology #

The effbot.org site runs on a custom Django application, with support for dynamic page rendering, in-browser editing, Subversion integration for software releases, etc.

I don’t plan to release the application at this time (except for maybe some support libraries). If you need something similar, I suggest using a real Wiki, such as MoinMoin.
