The encodings Package

cp1140

(New in 2.2) CP 1140 (EBCDIC 8-bit Western Europe, with Euro sign) codec. This codec is used to convert to and from Unicode strings.

idna

(New in 2.3) IDNA codec. This codec can be used to convert between Unicode strings and non-ASCII domain names, encoded as IDNA strings. [more (dead link)]

koi8_u

(New in 2.3) KOI8-U (Ukranian) codec for Unicode. [more (dead link)]

palmos

(New in 2.3) PalmOS 3.5 codec for Unicode. [more (dead link)]

punycode

(New in 2.3) Punycode IDNA codec. This is used to encode individual labels in an IDNA domain name (also see idna). [more (dead link)]

string_escape

(New in 2.3) Python string escape codec. This encoding replaces non-printable characters with Python-style string escapes. [more (dead link)]

utf_7

(New in 2.2) UTF-7 codec for Unicode.
