The effbot.exe/imageview Utility

April 29, 2003 | Fredrik Lundh

This effbot.exe utility is a simple image viewer, based on the Python Imaging Library.

[download] [user’s guide]

Download #

To run this effbot.exe application, you need to install the following packages, in order:

  • effbot-1.0.15-20030711.exe
  • pil114-1.1.4b1.effbot
  • imageview-0.0.3.effbot

(Newer versions may be available from the effbot.org downloads site).

Once you’ve installed effbot.exe/effnews, you should be able to install the pil114.effbot and imageview.effbot packages simply by downloading them and opening the files. If you’re using IE, you can open the files directly from the web page
