Announcing PIL 1.1.7 alpha 2

The second alpha for PIL 1.1.7 (code name “raclette”) is now available. Source code is available from Subversion and Mercurial repositories:

http://svn.effbot.org/public/tags/pil-1.1.7a2-20090328/

http://hg.effbot.org/pil-2009-raclette/src/pil-117a2-20090328/

You can also get source archives via the hg.effbot.org link (look for “download »”).

Windows binaries for Python 1.5.2, 1.6, 2.0, 2.1, 2.2, 2.3, 2.4, 2.5 and 2.6 can be found here:

http://effbot.org/downloads/#pil

This release can be considered as “pretty much feature complete”; what remains is some internal cleanup work, which probably won’t affect the public API or the semantics (but I still reserve the right to tweak things, hence the alpha label).

Bugs and other issues can be reported to the image-sig mailing list, or to the issue tracker available via the hg.effbot.org link above.
