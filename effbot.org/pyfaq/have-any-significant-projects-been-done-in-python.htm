Have any significant projects been done in Python?

Lots of projects use Python, for everything from commercial product development to testing and integration. To get an idea of what people are doing with Python, see this collection of Python success stories:

http://www.pythonology.org/python/success (dead link)

Consulting the proceedings for past Python conferences will also reveal contributions from many different companies and organizations.

High-profile Python projects include BitTorrent, the Mailman mailing list manager and the Zope application server. Several Linux distributions, including Ubuntu (dead link) and Red Hat, have written part or all of their installer and system administration software in Python. The “One Laptop Per Child” project uses Python for the user interface. Companies that use Python internally include Google, Yahoo, and Industrial Light & Magic.
