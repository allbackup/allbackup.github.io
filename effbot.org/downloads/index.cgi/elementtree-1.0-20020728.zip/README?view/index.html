======================
The elementtree module
======================

This kit contains a separate distribution of the ElementTree
data structure from the xmltoys library.

For more information on this library, see:

    http://effbot.org/zone/element-index.htm

The modules are designed to work with Python 2.1 and newer.  The
core tree implementation and the SimpleXMLTreeBuilder class also
works under 1.5.2.
