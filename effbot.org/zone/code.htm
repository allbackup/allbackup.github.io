Source Code Repositories

Fredrik Lundh | March 14, 2010

Most effbot.org code is available from one of two source code repositories.

The Subversion repository primarily contains stable code and “vendor drops” for modules available from the downloads page. For more information on the code available from this repository, see this page.

The Mercurial repository contains various works in progress.
