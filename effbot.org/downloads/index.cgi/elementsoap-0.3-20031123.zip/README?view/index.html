=======================
The elementsoap toolkit
=======================

This kit contains an early release of the ElementSoap toolkit, a
light-weight SOAP implementation based on the ElementTree library.

For more information on Elements, Element Trees, and this toolkit,
see:

    http://effbot.org/zone/element-index.htm
    http://effbot.org/zone/element-soap.htm
    http://effbot.org/zone/element-google.htm
