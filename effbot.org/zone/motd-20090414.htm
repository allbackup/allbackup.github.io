Announcing PIL 1.1.7 beta 1

Note: PIL 1.1.7 final coming soon. Stay tuned.

The first beta for PIL 1.1.7 (code name “raclette”) is now available. Source code is available as a tarball, and via Subversion and Mercurial repositories:

http://effbot.org/downloads/Imaging-1.1.7b1.tar.gz

http://svn.effbot.org/public/tags/pil-1.1.7b1-20090412/

http://hg.effbot.org/pil-2009-raclette/src/pil-117b1-20090412/

Windows binaries for Python 2.5 and 2.6 can be found here:

http://effbot.org/downloads/#pil

This release adds read support for interlaced PNG files, from Conrado Porto Lopes Gouvêa, and the usual collection of minor tweaks and fixes. The result should be pretty close to release candidate status.

Bugs and other issues can be reported to the image-sig mailing list, or to the issue tracker available via the hg.effbot.org link above.
