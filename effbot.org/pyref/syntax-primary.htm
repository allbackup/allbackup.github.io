Syntax: primaries

In the Python syntax, primaries represent the most tightly bound operations of the language. A primary is one of:

  • Atoms (syntax-atom), which is one of:
    • identifiers
    • literals
    • parenthesized-forms
    • list-displays
    • generator-expressions
    • dictionary-displays
    • string-conversions
  • attribute-references
  • subscriptions
  • slicings
  • calls
