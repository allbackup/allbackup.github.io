Is there a newsgroup or mailing list devoted to Python?

There is a high-traffic newsgroup, comp.lang.python, and a mailing list, python-list. The newsgroup and mailing list are gatewayed into each other — if you can read news it’s unnecessary to subscribe to the mailing list (and if you prefer using a newsreader, you can use gmane.org even if you don’t have access to a regular news feed).

Announcements of new software releases and events can be found in comp.lang.python.announce, a low-traffic moderated list that receives a few postings per day. It’s available as the python-announce mailing list.

More info about other mailing lists and newsgroups can be found at http://www.python.org/community/lists.html (dead link).
