The Pytte Intermediate Format

November 4, 2002 | Fredrik Lundh

The pytte1 interpreter cannot yet parse Python-like source code. Instead, it reads code in an intermediate XML format, using the built-in XML parser. Here’s the canonical “hello, world” program in a Python-like language:

print "hello, world!"

And here’s the same program in pytte1 XML:

 <module>
  <body>
   <print-newline>
    <const type="string">hello, world!</const>
   </print-newline>
  </body>
 </module>

(No, you’re not supposed to write pytte1 XML files by hand.)

To convert Python-like source code to this format, use the py2pyx converter (this is a Python program, running under CPython 2.2).

Note that the py2pyx converter supports a much larger subset of real Python than the current pytte1 code generator can handle.
