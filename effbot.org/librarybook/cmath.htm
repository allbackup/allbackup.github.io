The cmath module

This module contains a number of mathematical operations for complex numbers.

Example: Using the cmath module 
# File: cmath-example-1.py

import cmath

print "pi", "=>", cmath.pi
print "sqrt(-1)", "=>", cmath.sqrt(-1)


pi => 3.14159265359
sqrt(-1) => 1j
