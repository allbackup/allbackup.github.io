id

id(object)

Returns the identity of an object. This is an integer (or long integer) which is guaranteed to be unique and constant for this object during its lifetime. Two separate objects with non-overlapping lifetimes may have the same id value.

(Implementation note: in the current version, id returns the address of the object.)
