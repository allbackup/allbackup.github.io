list

list([sequence])

Creates a new list.

Returns a list whose items are the same and in the same order as sequence’s items. sequence may be either a sequence, a container that supports iteration, or an iterator object. If sequence is already a list, a copy is made and returned, similar to sequence[:]. For instance, list('abc') returns ['a', 'b', 'c'] and list((1, 2, 3) ) returns [1, 2, 3]. If no argument is given, returns a new empty list, [].
