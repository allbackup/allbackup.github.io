Syntax: Statements

Simple Statements #

Simple statements are comprised within a single logical line. Several simple statements may occur on a single line separated by semicolons. The syntax for simple statements is:

  • expression-statements
  • assert
  • assignments
  • pass
  • del
  • print
  • return
  • yield
  • raise
  • break
  • continue
  • import
  • global
  • exec
